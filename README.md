
## Prognozowanie wyników inwestycyjnych z kwantyfikacją ryzyka ##

Projekt zakłada stworzenie aplikacji desktopowej o nazwie RiskFolio, służącej do budowania wirtualnego portfela aktywów i symulowania jego przyszłej wartości w oparciu o dane historyczne i prognozowanie metodą Monte Carlo.
